🔄
각색 모드

PRO
🌐

👥
IP 라이브러리

💬
소리와 대화

왜 다른 도구와 다른가

🇰🇷 한국어 humanizer

AI 티 6패턴 자동 검출 — 번역투, 격언체, 관념어, 깔끔한 대구문, 메타포 과정리. Tier 1/2/3 단어 시스템.

🎬 배급사 다중 타겟 리뷰

10대/30대 직장인/40대 가족 — 타겟별 별도 리뷰. 알바 리서치 양식 그대로. 제목 평가, 다른 제목 추천 포함.

🔄 50번+ 각색 반복

영화·드라마는 수십 번 고친다. 자연어 디렉션 + 자동 버전 관리 + 변경 이력 추적.

📐 13개 장르 표준 양식

장르마다 다른 한국 실무 양식 자동 적용. 폰트, 페이지, 분량, 씬 헤딩까지.

🧠 쓰면서 똑똑해짐

좋아한 패턴/반려한 패턴 누적 학습. 작가 프로필 자동 반영.

📝 워드/한글 룩앤필

편집 영역은 함초롬바탕 11pt. 본문 폰트 8종 즉시 전환. 작가가 익숙한 양식 그대로.
