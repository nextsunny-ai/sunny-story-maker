# SUNNY Story Maker v1.0

> 한국어 시나리오·대본 작가를 위한 AI 협업 창작 툴.
> 13개 장르(드라마/영화/숏드라마/애니/웹툰/다큐/웹소설/뮤지컬/유튜브/전시/게임/예능 등) 한국 실무 표준 포맷 지원.

---

## 🚀 빠른 시작

### 1. 실행
```bash
C:\SUNNY_Story_Maker\run.bat
```
첫 실행 시 가상환경 + 의존성 자동 설치 (1~2분).
브라우저 자동 오픈 (http://localhost:8501).

### 2. API 키 등록
사이드바 → **설정** → `ANTHROPIC_API_KEY` 입력
또는 `.env`에 직접:
```
ANTHROPIC_API_KEY=sk-ant-...
```

### 3. 작가 프로필 등록 (권장)
사이드바 → **작가 프로필** → 신규 등록 → 활성화
- 프로필 등록하면 모든 모드(AI 기획/공동 집필/평가/각색)에서 자동으로 작가 스타일 반영
- 안 해도 OK — 내장 기본 노하우(humanizer/Tier/13장르)는 항상 적용됨

## 🎯 7가지 모드

| 모드 | 용도 |
|------|------|
| 🪄 **AI 기획** | 한 줄 아이디어 → 작가 에이전트가 처음부터 끝까지 |
| ✏️ **공동 집필** | 사용자 정보 입력 → 5단계 프로세스 (장르마다 단계/필드 자동 변경) |
| 🔍 **평가 (★)** | 첨부 → **다중 타겟 리뷰** (배급사 알바 문항지 양식) |
| 🔄 **각색** | 같은 장르 50회+ 반복 / 다른 장르 변환 |
| 🌐 **OSMU** | 한 IP를 13장르 매트릭스로 |
| 👥 **라이브러리** | 작가 본인 캐릭터/세계관 IP 관리 |
| 💬 **소리와 대화** | 보조작가처럼 자유로운 채팅 (옵션) |

## 🎬 13개 지원 장르

A. TV 드라마 / B. 영화 / C. 숏드라마 / D. 극장 애니 / E. 애니 시리즈 /
F. 웹툰 / G. 다큐멘터리 / H. 웹소설 / I. 뮤지컬·공연 /
J. 유튜브·숏폼 / K. 전시·체험형 / L. 게임 / M. 예능

각 장르마다:
- 5종 세트 자동 적용: 시장 현황 / 핵심 전략 / 선결정 사항 / 실패 요인 / AI투 제거
- 한국 실무 표준 포맷 자동 (페이지/폰트/분량/씬 헤딩)
- 장르별 단계+입력 필드 동적 변경

## 📦 핵심 기능 디테일

### 평가 모드 (★ 단독 셀링 가능)
- 배급사 알바 리서치 문항지 양식 그대로
- 타겟 3개 별도 평가 (10대/20대 도시 여성/30대 직장인/40~50대 가족 등)
- **12개 정형 문항**:
  1. 제목 평가 / 2. 제목 추천 / 3. 1차 반응 / 4. 좋았던 점 Top3 / 5. 별로였던 점 Top3
  6. 캐릭터별 호감도 / 7. 대사 반응 / 8. 클라이맥스 / 9. 결말 / 10. 고칠 점 Top5
  11. 추천 의향 / 12. 한 줄 평
- 헐리우드 코버리지 표준 등급 (PASS / CONSIDER / RECOMMEND)
- 14개 평가 카테고리 점수 (컨셉/캐릭터/스토리/플롯/씬/주제/장르/페이싱/톤/대사/시장성/문체/포맷/제목)

### 각색 모드 (50회+ 반복)
- 같은 장르 내 자연어 디렉션 ("S#3 무게 더해", "도윤 대사 줄여")
- 부분 수정 (전체/특정 씬/특정 회차/특정 캐릭터)
- 자동 버전 관리 (v1, v2, v3 ... 무한)
- 변경 이력 + 부모 버전 추적
- 다른 장르 변환 추천 (사용자 명시 요청 시만)

### humanizer (AI투 검출)
- **6패턴 자동 검출**: 번역투 "의" 이중 / 관료적 피동 / 격언체 / 관념어 / 깔끔한 대구문 / 메타포 과정리
- **Tier 1/2/3 단어 시스템** (항상 교체 / 반복 시 교체 / 밀도 따라 교체)
- 색깔 하이라이트 + 수정 제안
- 점수화 (0=깨끗, 100=AI 티 강함)

### 작가 프로필 + 누적 학습 (★)
- **자기 스타일/선호/안티패턴/비유 체계** 등록
- 모든 모드에서 자동 컨텍스트 반영
- **사용하면서 누적 학습**: 좋아한 패턴 / 반려한 패턴 / 자주 받는 디렉션 자동 기록
- 다음 작업 시 누적된 노하우가 system prompt에 자동 통합 → **쓸수록 똑똑해짐**

## 💡 내장 기본 노하우 (사용자가 비워둬도 항상 적용)

- **humanizer 6패턴** + Tier 1/2/3 단어 시스템
- **13장르 매뉴얼** (시장 현황 / 전략 / 선결정 / 실패 요인 / AI투)
- **5단계 프로세스** (의뢰 분석 → 트리트먼트 → 시놉시스 → 초안 → humanizer 검증)
- **캐릭터 비유 체계 방법론** (직업/취미 기반)
- **이름 가리기 / 대사 교환 / 감정 변환 3테스트**
- **한국 대사 룰** (숫자 한글 / 사투리 활용 / 호명 룰)
- **자기 검증 체크리스트** + 2차 자가 감시 루프
- **헐리우드 코버리지 표준** (PASS/CONSIDER/RECOMMEND + 14 카테고리)
