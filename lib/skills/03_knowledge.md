# 소리(SORI) 지식베이스 — 시나리오·드라마 이론 & 한국어 글쓰기

> 작가의 판단 = 이 지식 위에 선다. 매 시나리오 작성 시 = 이 문서 참조.

## A. 시나리오 구조 이론

### 1. 3막 구조 (Syd Field, "Screenplay")
- **1막 Setup (1~30 페이지)**: 주인공·세계 소개. Inciting Incident (10~15 페이지). Plot Point 1 (~25%).
- **2막 Confrontation (30~90 페이지)**: 갈등 심화. Midpoint (~50%). Plot Point 2 (~75%).
- **3막 Resolution (90~120 페이지)**: Climax · Denouement.

### 2. 5막 구조 (Robert McKee, "Story")
- **Inciting Incident** → 주인공 일상 깨짐
- **Progressive Complications** → 점점 심화
- **Crisis** → 결정의 순간
- **Climax** → 행동·결과
- **Resolution** → 새 균형

### 3. Save the Cat (Blake Snyder) 15 비트
- Opening Image · Theme Stated · Setup · Catalyst · Debate · Break Into Two · B Story · Fun and Games · Midpoint · Bad Guys Close In · All Is Lost · Dark Night of the Soul · Break Into Three · Finale · Final Image

### 4. 기·승·전·결 (한국·동양)
- **기 (起)**: 도입. 인물·배경·갈등 씨앗.
- **승 (承)**: 발전. 갈등 심화.
- **전 (轉)**: 반전. ★ 가장 중요. 시청자 예측 깨기.
- **결 (結)**: 결말. 갈등 해소·여운.

### 5. Hero's Journey (Joseph Campbell, "The Hero with a Thousand Faces") 12 단계
1. Ordinary World · 2. Call to Adventure · 3. Refusal · 4. Meeting the Mentor · 5. Crossing the Threshold · 6. Tests/Allies/Enemies · 7. Approach · 8. Ordeal · 9. Reward · 10. The Road Back · 11. Resurrection · 12. Return with Elixir

## B. 캐릭터 이론

### 1. Want vs Need (Robert McKee)
- **Want**: 캐릭터가 표면적으로 원하는 것
- **Need**: 캐릭터가 진짜로 필요한 것
- 캐릭터 아크 = Want 추구 → Need 깨달음

### 2. 캐릭터 매트릭스 (4분면)
| | 외향 | 내향 |
|---|---|---|
| **행동형** | 리더·전사 | 관찰자·전략가 |
| **사고형** | 변호사·정치가 | 학자·예술가 |

### 3. 적대자 (Antagonist) 깊이
- 단순한 악역 X
- 자기만의 Want·Need
- 주인공보다 강한 면 1개 이상

### 4. 조연 활용
- 주인공 거울 (대비 또는 유사)
- 정보 제공 (Exposition)
- 톤 조절 (코믹 릴리프)

## C. 대사·문체 이론

### 1. Subtext (서브텍스트)
- 대사 = 표면 의미 / 의도 = 행간
- 캐릭터가 "솔직하지 않을 때" Subtext 강해짐
- 영화 = Subtext 우선 / 드라마 = 직설+Subtext 혼합

### 2. "On the Nose" 회피
- 캐릭터가 자기 감정을 직접 말 X
- 행동·표정·다른 화제로 우회

### 3. 대사 리듬
- 짧은 대사 ↔ 긴 대사 교차
- 침묵·휴지부 활용
- 1~2 문장 → 한 단락 → 짧은 응답

## D. 한국어 글쓰기 (★ humanizer 핵심)

### 1. 번역투 회피

| 번역투 | 자연스러움 |
|---|---|
| 그것을 ~한다 | ~한다 |
| ~에 의해 (수동) | ~가 ~한다 (능동) |
| ~의 ~의 ~의 (반복) | 줄이기 |
| 그러나 / 하지만 (영어 직역) | 그런데 / 근데 (구어) |
| ~을 위해서 | ~려고 / ~려 |

### 2. 한글 발음 룰
- 받침 없는 음절 + 자음 시작 = 자연스러운 흐름
- 비슷한 자음 반복 = 운율
- 어색한 종성 = 읽기 어려움

### 3. rule-of-three 강박 회피
- 3개씩 묶기 = 작가 강박. 자연스러운 N개로.
- 사례: "조선의 요괴의 그림자의 끝" → "조선 요괴의 그림자, 그 끝"

### 4. 한국 드라마·영화 화법 (참고)
- 김은희: 사극+미스터리. 직설적·강한 대사.
- 박찬욱: 시네마틱. Subtext 강함.
- 봉준호: 일상+풍자. 자연스러운 한국어.
- 박완서·황석영: 한국어 문체의 정수.
