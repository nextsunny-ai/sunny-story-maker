## A. 시나리오 사이트 (영문 원본)

### 1. Script Slug (https://www.scriptslug.com)
- 영화 시나리오 PDF 무료 (수천 편)
- 카테고리·장르·작가별 검색
- 매일 1편 = 시나리오 분석 추천

### 2. Simply Scripts (https://www.simplyscripts.com)
- 영화·드라마 시나리오 라이브러리
- 단편·장편·TV 다 있음

### 3. IMSDB (https://www.imsdb.com)
- Internet Movie Script Database
- 클래식·최신 시나리오

### 4. Daily Script (https://www.dailyscript.com)
- 매일 한 편 시나리오 추천
